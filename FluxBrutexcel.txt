Ce tableau présente des données numériques associées à plusieurs villes pour des dates spécifiques. Voici une description simple de sa structure et de son contenu :

Structure du tableau
DATE : Cette colonne liste les dates pour lesquelles les données sont enregistrées. Par exemple :

Vendredi 1 Janvier 2016
Samedi 2 Janvier 2016
Dimanche 3 Janvier 2016
Colonnes de villes : Chaque colonne suivante représente une ville différente. Par exemple :

Bègles Rives d'Arcins
Bordeaux Intendance
Bordeaux Sainte Catherine

Mérignac

Données du tableau
Les lignes sous les dates contiennent des valeurs numériques qui représentent des mesures spécifiques pour chaque ville à une date donnée. Par exemple :
Le 1er janvier 2016, toutes les valeurs sont à 0 pour toutes les villes.
Le 2 janvier 2016, les valeurs varient, par exemple, 664 pour Bègles Rives d'Arcins, 351 pour Bordeaux Intendance, etc.
Le 3 janvier 2016, toutes les valeurs sont à nouveau à 0 pour toutes les villes.
